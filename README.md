\# PyCalc



\*\*PyCalc\*\* is a simple graphical calculator made with Python and Tkinter.  

It allows you to perform basic arithmetic operations with an easy-to-use GUI.



\## Features

\- Addition (+), subtraction (-), multiplication (\*), division (/)  

\- Decimal numbers support  

\- Clear and backspace buttons  

\- Press \*\*Enter\*\* key or click \*\*=\*\* to calculate  

\- Clean and colorful interface  



\## How to Run

1\. Make sure Python 3 is installed: \[Python Downloads](https://www.python.org/downloads/)  

2\. Open a terminal or command prompt in the project folder  

3\. Run the calculator:



```bash

python app.py
